# 🌊 Rising Waters: Flood Prediction System

## Overview

Floods are among the most devastating natural disasters. This project predicts flood risk using Machine Learning algorithms based on environmental and weather-related parameters.

## Features

- Predict flood risk instantly
- Flask web application
- User-friendly interface
- XGBoost model deployment
- Live deployment on Render

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Scikit-Learn
- XGBoost
- NumPy
- Pandas

## Machine Learning Models

| Model | Accuracy |
|-------|----------|
| Decision Tree | 69.94% |
| KNN | 84.57% |
| Random Forest | 90.28% |
| XGBoost | 93.24% |

## Installation

```bash
pip install -r requirements.txt
python app.py
```
